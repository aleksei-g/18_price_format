# 18_price_format

# Форматирование цены

Скрипт приводит входящую строку цены товара к более наглядному виду.
Например цена `'1999.99'` будет выглядеть `'1 999.99'`.

Скрипт можно использовать двумя способами:
* запуск из командной стоки:

```
python format_price.py 1999.99
```
* вызов модуля из другого скрипта:

```
from format_price import format_price
format_price(1999.99)
```
Так же в репозитории находится файл с тестами `tests.py`.
